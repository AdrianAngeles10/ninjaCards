# ninja card game

# Project description

This project has a player at the bottom of the map to have a sword and whenever a card appears you must slice it. There are random cards spawning randomly on top of the screen. Once the player hits the card the card turns backwards and will get added to your point system.

# tools used

I did all my developing in Godot so the programming laguages used were just the godot scripting language. I also used free sprites from itch.io such as https://luizmelo.itch.io/martial-hero (for samurai sprite).

# how to run

1. clone the repository
2. open root of repository in godot
3. go into proejct and hit play
